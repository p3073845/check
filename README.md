# User Management Application

A modern React-based user management system built with Vite.

## Requirements

- Node.js 16.x or higher
- npm 8.x or higher

## Installation

```bash
npm install
```

## Development

Start the development server:

```bash
npm run dev
```

## Build

Create a production build:

```bash
npm run build
```

## Dependencies

### Core
- React 19.1.0
- React Router DOM 7.5.1
- Redux Toolkit 2.7.0
- React Redux 9.2.0

### UI & Utilities
- Bootstrap 5.3.5
- Axios 1.8.4
- Date-fns 2.30.0
- jQuery 3.7.1
- Sonner 1.4.0
- Tippy.js 6.3.7

## Development Dependencies
- Vite 6.3.1
- ESLint 9.22.0
- TypeScript types for React
